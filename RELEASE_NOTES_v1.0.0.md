# Release Notes v1.0.0

This release supports manuscript submission for the MIMIC-IV v3.1 retrospective cohort study of documented psychiatric comorbidity, early ICU delirium, and post-discharge outcomes.

Final result hierarchy:

- Mortality: primary_mortality_v1_1.
- Readmission cause-specific Cox/time-varying analyses: validated 02_readmission_outcomes.
- Readmission standardized CIF: 02_readmission_outcomes_v1_2 only.
- Sensitivity analyses: 03_sensitivity_analyses.

No MIMIC patient-level data are included.
