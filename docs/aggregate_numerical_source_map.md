# Aggregate Numerical Source Map

This public release maps manuscript-level numerical statements to aggregate, non-patient-level source files.

- Cohort and four-group counts: `results/integrated/final_results_summary.csv` and analysis-data freeze manifests.
- Mortality Cox estimates: `results/mortality/cox_joint_exposure_models_v1_1.csv`.
- Mortality standardized 365-day risks and additive interaction: `results/mortality/fixed_horizon_365d_standardized_risk.csv` and `results/mortality/fixed_horizon_365d_additive_interaction.csv`.
- Readmission cause-specific HRs: `results/readmission/readmission_90d_cause_specific_cox.csv`, `results/readmission/icu_readmission_365d_cause_specific_cox.csv`, and `results/readmission/competing_death_cause_specific_cox.csv`.
- Readmission standardized CIFs: `results/readmission/standardized_90d_readmission_cif_v1_2.csv` and `results/readmission/standardized_365d_icu_readmission_cif_v1_2.csv`.
- Readmission additive interaction: `results/readmission/additive_interaction_readmission_90d_v1_2.csv` and `results/readmission/additive_interaction_icu_readmission_365d_v1_2.csv`.
- Figure source data: `figures/main/figure_source_data.csv` and `figures/supplementary/source_data.csv`.
