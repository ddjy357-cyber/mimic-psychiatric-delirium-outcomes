# Zenodo Manual Steps

No `ZENODO_TOKEN` was available during local packaging. Do not invent a DOI.

Manual options:

1. Log in to Zenodo.
2. Connect GitHub.
3. Enable the `mimic-psychiatric-delirium-outcomes` repository.
4. Create the GitHub `v1.0.0` release.
5. Wait for Zenodo archival.
6. Copy the DOI and DOI URL into the manuscript Data availability statement.

Alternatively, create a Zenodo draft upload manually and use "Get a DOI now" to reserve a DOI before final publication.
